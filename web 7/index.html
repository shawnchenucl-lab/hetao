Design Concept / 設計概念

有序之光，聯通之門

Light of Order, Gateway of Collaboration

河套的核心價值不在「邊界」，而在「連接」：連接深圳與香港，連接內地創新製造體系與香港國際科研資源。

夜間照明不作炫目地標，而作清晰、高效、可信賴的科技園門戶。

以 克制、精確、低眩光、重層次 的外立面光環境，支撐高人流旅檢建築的入口識別、流線引導與穩定運營。

Color Temperature Strategy / 色溫策略

選擇 3500K：在清晰效率與克制舒適之間取得平衡

旅檢建築需要清晰、可信賴、利於識別的公共光環境；同時，河套科技園的夜間門戶不宜過冷或過度商業化。 因此，本方案建議以 3500K 作為主體色溫，在 4000K 的透亮效率與 3000K 的柔和克制之間建立更穩定的中性平衡。

Scheme 01 / 主推方案

Vertical Order

底部連續線性上照與豎向光序

Recommended

以底部橫向 continuous linear light 向上洗亮格柵，形成受控豎向光序。

  • 燈具設於格柵最下方水平燈槽，光線向上擦亮豎向構件。
  • 每米 8 pixel 控制單元，DMX 分段控制，適合精細亮度與節奏管理。
  • 參考參數：12W/m，20×40° optic，3500K 均衡中性白。
  • 豎向光序象徵深港之間人員、知識、科研資源與創新要素的有序流動。
Technical Note

以格柵底部水平燈槽形成連續上照，通過槽內反射與分段控制建立豎向光序；重點是低眩光、可控亮度與構造一體化，而非外露燈帶。

Scheme 02 / 備選方案

Horizontal Layer

層間下照與水平秩序

Alternative

通過層間頂部或樓板下方的隱藏線性燈，向下形成受控的水平光層。

  • 線性燈隱藏於層間或檐口內側，避免形成外露輪廓燈。
  • 向下擦亮局部立面，建立清晰、穩定的水平層級。
  • 相比方案一更整體、更安靜，但需要嚴格控制亮度與連續性。
Technical Note

以層間頂部或樓板底部的隱藏線性下照形成水平光層，重點是控制亮度、避免每層外露勾邊，讓建築呈現穩定而克制的公共建築秩序。

Scheme 03 / 備選方案

Inner Glow + Gateway Frame

內透光與門戶框架

Alternative

以入口玻璃界面、門廊頂棚與基座內透光，形成清晰可達的夜間門戶。

  • 上部立面保持低亮度，只做局部收邊。
  • 入口、排隊、通行界面作主要識別。
  • 以門廊及連廊節點補足城市可見度。
Technical Note

以入口玻璃界面、門廊頂棚與基座內透光建立夜間可讀性，讓主要光環境服務旅客到達、排隊、識別與通行，而不是上部立面表演。
