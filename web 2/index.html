Design Concept / 設計概念

有序之光，聯通之門

Light of Order, Gateway of Collaboration

河套的核心價值不在「邊界」，而在「連接」：連接深圳與香港，連接內地創新製造體系與香港國際科研資源。

夜間照明不作炫目地標，而作清晰、高效、可信賴的科技園門戶。

以 克制、精確、低眩光、重層次 的外立面光環境，支撐高人流旅檢建築的入口識別、流線引導與穩定運營。

Scheme 01 / 主推方案

Vertical Order

底部連續線性上照與豎向光序

Recommended

以底部橫向 continuous linear light 向上洗亮格柵，形成受控豎向光序。

  • 燈具設於格柵最下方水平燈槽，光線向上擦亮豎向構件。
  • 每米 8 pixel 控制單元，DMX 分段控制，適合精細亮度與節奏管理。
  • 參考參數：12W/m，20×40° optic，4000K 中性白。
  • 豎向光序象徵深港之間人員、知識、科研資源與創新要素的有序流動。
Technical Note

重點是「底部橫向連續線性上照」與「DMX 分段控制」，不是點狀燈或外露輪廓線。

Scheme 02 / 備選方案

Point Downlight Rhythm

格柵間點式下照

Alternative

在格柵與格柵之間布置小筒燈式點光源，向下形成受控光斑。

  • 點光源藏於格柵深度內，避免直視眩光。
  • 下照形成節奏化光斑，強化立面秩序。
  • 相比方案一更輕、更離散，夜間識別較克制。
Technical Note

重點是「點狀下照」：燈位在格柵間，光往下落，不做連續線性上照。

Scheme 03 / 備選方案

Inner Glow + Gateway Frame

內透光與門戶框架

Alternative

以入口玻璃界面、門廊頂棚與基座內透光，形成清晰可達的夜間門戶。

  • 上部立面保持低亮度，只做局部收邊。
  • 入口、排隊、通行界面作主要識別。
  • 以門廊及連廊節點補足城市可見度。
Technical Note

重點在城市外部可見的入口亮度、門廊框架與人流導向，而非上部立面表演。
